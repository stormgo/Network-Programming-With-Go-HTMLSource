MonQueST - Lecturing

MONash QUEstionnaire Series on Teaching

Answer the following questions ONLY in terms of THIS LECTURER'S LECTURING in THIS SUBJECT. *

This is not an official Monquest survey. News of the redundancies came too late for one to be organised. Nevertheless, if you can fill this in (just once) then I would find it most useful. Please be honest in your answers. The questionnaire is anonymous - Jan

Each question describes an aspect of teaching or learning that you could expect to have observed or experienced during ANY SINGLE LECTURE. Answer each by recalling this lecturer's lectures. For HOW MANY of these lectures did the described aspect occur? Answer each question independently, without referring to aspects of teaching or learning described in any other question.

The responses are defined as follows:

The description was true for...
  • ALL OR ALMOST ALL of the lectures
  • MOST of the lectures
  • ABOUT HALF of the lectures
  • ONLY SOME of the lectures
  • VERY FEW OR NONE of the lectures

There are two circumstances in which you will be UNABLE to answer a question. First, the description is ENTIRELY INAPPROPRIATE as something that could have been expected to occur in this subject. Instances of this would be rare. Second, you might have ATTENDED TOO FEW lectures to be able to make proper judgements. If either of these circumstances apply, you should use one of the cannot answer responses.

Please indicate your answers by clicking one response for each question (note that there is room for written comment at the end).
